## 📥 تثبيت البرنامج (للصيدلاني)

### الخطوة 1 — تنزيل البرنامج
اذهب إلى [Releases](../../releases/latest) ونزّل آخر إصدار:
```
medtracker-vX.X.X-windows.zip
```

### الخطوة 2 — فك الضغط
فك الضغط في أي مجلد. ستجد ملفين **يجب أن يبقيا في نفس المجلد**:
```
📁 medtracker/
├── medtracker.exe          ← البرنامج الرئيسي
├── better_sqlite3.node     ← محرك قاعدة البيانات (لا تحذفه)
└── تعليمات-التثبيت.txt
```

### الخطوة 3 — تشغيل البرنامج
شغّل `medtracker.exe` — سيفتح المتصفح تلقائياً

### الخطوة 4 — إعداد البوت (مرة واحدة فقط)

ستظهر صفحة إعداد تطلب منك:

| المطلوب | كيف تحصل عليه |
|--------|--------------|
| **توكن البوت** | افتح تليجرام ← @BotFather ← `/newbot` |
| **Chat ID** | افتح تليجرام ← @userinfobot ← أرسل أي رسالة |
| **مفتاح Groq AI** | سجّل مجاناً على [console.groq.com](https://console.groq.com) ← API Keys |

بعد الإعداد يبدأ البوت مباشرة ✅

## 🤖 أوامر البوت في تليجرام

| الأمر | الوظيفة |
|-------|---------|
| `/start` | عرض قائمة الأوامر |
| `/scan` | مسح فاتورة نوبكو أو علبة دواء (صورة أو PDF) |
| `/add اسم باتش تاريخ كمية` | إضافة دواء يدوياً |
| `/list` | عرض المخزون كاملاً مع الألوان |
| `/check` | أدوية تنتهي هذا الشهر |
| `/check week` | أدوية تنتهي هذا الأسبوع |
| `/report` | تقرير فوري |

> 📊 يصلك تقرير تلقائي كل يوم الساعة 9 صباحاً

## 🏗️ للمطورين — بناء من المصدر

```bash
# متطلبات: Node.js 18+
git clone https://github.com/fahadx7/nupco-pharma
cd nupco-pharma
npm install

# تشغيل للتطوير
npm run dev

# بناء الـ exe (يتطلب Windows)
npm run build
# الناتج في dist/medtracker.exe + better_sqlite3.node
```

### إصدار نسخة جديدة (GitHub Actions يبني تلقائياً):
```bash
git tag v3.1.0
git push --tags
# GitHub Actions يبني الـ exe وينشره في Releases تلقائياً
```
