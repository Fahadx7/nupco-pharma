# 💊 نوبكو فارما — بوت تتبع صلاحية الأدوية

بوت تليجرام لتتبع انتهاء صلاحية أدوية نوبكو مع واجهة ويب للإدارة.

## المميزات

- 📸 مسح فواتير نوبكو وعلب الأدوية (صور + PDF)
- ➕ إضافة أدوية يدوياً
- 📋 عرض المخزون الكامل
- 🔍 فحص الأدوية القريبة من الانتهاء
- 📊 تقرير يومي تلقائي الساعة 9 صباحاً
- 🌐 واجهة ويب للإدارة

## التقنيات

- Node.js + Telegram Bot API
- Neon PostgreSQL
- Groq AI (مسح الصور)
- Express.js (واجهة الويب)

## التشغيل

```bash
# تثبيت المكتبات
npm install

# نسخ ملف البيئة وتعبئته
cp .env.example .env

# تشغيل البوت
npm start

# تشغيل واجهة الويب
npm run web
```

## متغيرات البيئة

انظر `.env.example`
